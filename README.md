# 🎮 休闲小游戏合集

7 款经典小游戏，纯原生 HTML5 + CSS3 + JavaScript 实现，零外部依赖，双击 `index.html` 即可游玩。

## 游戏列表

| 游戏 | 目录 | 简介 |
|------|------|------|
| 🐍 贪吃蛇 | `snake/` | 经典贪吃蛇，粒子特效 + 合成音效 |
| 🧱 打砖块 | `breakout/` | 多关卡打砖块，挡板 + 弹球 |
| 🔢 2048 | `game2048/` | 数字合并益智游戏 |
| 💎 消消乐 | `match3/` | 宝石三消，连锁倍率加分 |
| 🃏 翻牌记忆 | `memory/` | 多难度翻牌配对记忆游戏 |
| 🟦 俄罗斯方块 | `tetris/` | 经典方块消行，含幽灵块预览 |
| 🐭 打地鼠 | `whack/` | 30秒限时打地鼠，连击加倍 |

## 运行方式

直接双击任意游戏目录内的 `index.html`，或启动本地服务器后访问主页：

```bash
# Python
python -m http.server 8080

# Node.js
npx serve .
```
