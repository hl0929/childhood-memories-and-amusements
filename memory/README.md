# 🃏 翻牌记忆

翻牌配对记忆游戏，基于 DOM + CSS 3D 翻转实现，零外部依赖。

## 特色

- 三种难度：4×4（简单）/ 4×6（中等）/ 5×6（困难）
- CSS 3D perspective 翻牌动画
- 计时器 + 翻牌次数实时显示
- 各难度独立保存最佳翻牌次数（localStorage）
- 32 种 Emoji 随机配对，每局不重复

## 操作

| 操作 | 方式 |
|------|------|
| 翻牌 | 点击卡牌 |
| 切换难度 | 点击难度按钮 |
| 重开 | 点击重新开始 |
