# 🔢 2048

数字合并益智游戏，基于 DOM + CSS 实现，零外部依赖。

## 特色

- 标准 4×4 网格，上下左右四方向滑动合并
- 新方块出现动画（pop 弹出效果）
- 达成 2048 后可继续挑战更高数字
- localStorage 持久化最高分
- 键盘 / 触屏滑动双重输入

## 操作

| 操作 | 方式 |
|------|------|
| 移动 | 方向键 / WASD |
| 重开 | R |
| 移动端 | 屏幕滑动 |
