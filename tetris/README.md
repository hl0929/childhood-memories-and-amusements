# 🟦 俄罗斯方块

经典俄罗斯方块，基于 HTML5 Canvas 实现，零外部依赖。

## 特色

- 标准 10×20 网格，七种标准方块（I/O/T/J/L/S/Z）
- 幽灵块：半透明预览方块落点
- 下一个方块预览面板
- Wall kick：旋转时自动修正贴墙偏移
- 消行得分：单行 100 / 双行 300 / 三行 500 / 四行 800（乘以关卡倍率）
- 每 10 行升一关，最大下落间隔 80ms
- localStorage 持久化最高分
- 移动端虚拟按键 + 触屏滑动手势

## 操作

| 操作 | 按键 |
|------|------|
| 左右移动 | 方向键 / A·D |
| 旋转 | 上方向键 / W |
| 软降 | 下方向键 / S |
| 硬降（立即落下）| 空格 |
| 暂停 / 继续 | P |
| 重开 | R |
| 移动端 | 触屏滑动 / 虚拟按键 |
