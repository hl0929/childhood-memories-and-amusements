# 🧱 打砖块

经典打砖块游戏，基于 HTML5 Canvas 实现，零外部依赖。

## 特色

- 多关卡：清空砖块自动进入下一关，球速随关卡提升
- 双血量砖块：3 关以上第一排砖块需击打 2 次才能消除
- 挡板支持鼠标 / 触屏跟随，以及键盘左右控制
- 根据击球位置改变反弹角度，球速始终恒定
- 砖块消除时触发粒子爆炸特效
- localStorage 持久化最高分
- 3 条命，掉球扣命，命尽结束

## 操作

| 操作 | 方式 |
|------|------|
| 移动挡板 | 鼠标移动 / 方向键 A·D |
| 发球 | 空格 / 点击画布 |
| 暂停 / 继续 | P |
| 重开 | R |
| 移动端 | 触屏拖动挡板 |
