# 🐭 打地鼠

30 秒限时打地鼠游戏，基于 DOM + CSS 动画实现，零外部依赖。

## 特色

- 9 个洞穴，地鼠随机出没
- 炸弹（18% 概率）：点击扣 10 分并重置连击
- 连击系统：3 连击 ×2 分，5 连击 ×3 分
- 随时间推移地鼠停留时间缩短、出现间隔加快
- 得分浮动提示动画
- localStorage 持久化最高分

## 操作

| 操作 | 方式 |
|------|------|
| 打地鼠 | 点击 / 触屏地鼠 |
| 开始游戏 | 点击开始按钮 |
