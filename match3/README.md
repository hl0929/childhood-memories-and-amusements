# 💎 消消乐

宝石三消游戏，基于 DOM + CSS 实现，零外部依赖。

## 特色

- 8×8 棋盘，7 种宝石 Emoji
- 连锁消除：一次交换触发多次连锁，倍率递增
- 初始棋盘保证无预置匹配，且至少有一个可用步骤
- 无效交换时宝石高亮提示，无路可走时自动洗牌
- 30 步内达到目标分数胜利，步数用尽失败
- localStorage 持久化最高分
- 点击选中 + 点击相邻 / 触屏滑动两种交换方式

## 操作

| 操作 | 方式 |
|------|------|
| 选中宝石 | 点击 |
| 交换宝石 | 点击相邻格 / 触屏滑动 |
| 重开 | 点击结果面板按钮 |
