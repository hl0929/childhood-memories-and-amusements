# 🐍 贪吃蛇

经典贪吃蛇，基于 HTML5 Canvas 实现，零外部依赖。

## 特色

- 蛇身渐变色 + 眼睛朝向 + 段间平滑连接
- 双食物类型：红色普通 (1分) 、金色稀有 (3分，15% 概率)
- 粒子特效：吃食物 / 死亡 / 新纪录时爆发粒子
- Web Audio API 合成音效，无需音频文件
- 每吃 5 个食物自动加速，最快 50ms/帧
- localStorage 持久化最高分，高 DPI 适配
- 移动端触屏滑动 + 虚拟方向键

## 操作

| 操作 | 按键 |
|------|------|
| 移动 | 方向键 / WASD |
| 暂停 / 继续 | P |
| 重开 | R |
| 移动端 | 屏幕滑动 / 虚拟方向键 |
